# Term Selection Matrices

Status: Draft for WG use
Date proposed: 07.06.2026

This note provides one structured selection matrix for each term in [[INCOSE-MENA-Arabic-Glossary]].

Use the process in [[Terminology-Adoption-Process]]. Scores should be from 1 to 5, then multiplied by the weight for each criterion.

This approach is used for all contested terms and style guide decisions. It ensures transparency and objectivity, and keeps the discussion away from personal preferences.

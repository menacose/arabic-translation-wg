## Source Definition

Development of to-be system or system elements in the presence of existing or legacy as-is system or system elements. A brownfield approach is usually used to extend, improve, or replace a system that is in use or to reuse system elements that will not be impacted by the desired changes. The new system architecture must take into account the existing system elements and functions, which impose constraints on the overall system definition.
